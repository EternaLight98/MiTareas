# MiTareas - Aplicación de Gestión de Tareas Personales

## Descripción
MiTareas es una aplicación web para la gestión de tareas personales que permite a los usuarios registrarse, iniciar sesión, y administrar sus tareas diarias.

## Características
- Registro de usuarios
- Inicio y cierre de sesión
- Gestión completa de tareas (crear, editar, eliminar, visualizar)
- Cierre automático de tareas vencidas
- Interfaz responsiva y amigable

## Requisitos
- Node.js (v14 o superior)
- npm (v6 o superior)

## Instrucciones de instalación

1. Descomprima el archivo ZIP en la ubicación deseada
2. Abra una terminal en la carpeta descomprimida
3. Ejecute el siguiente comando para instalar las dependencias:
   ```
   npm install
   ```
4. Inicie la aplicación con:
   ```
   npm start
   ```
5. Abra su navegador y vaya a: `http://localhost:3000`

## Tecnologías utilizadas
- HTML5, CSS3, JavaScript (ES6+)
- Node.js con Express para el servidor

## Estructura del proyecto
- `index.html`: Estructura principal de la aplicación
- `style.css`: Estilos y diseño responsivo
- `app.js`: Lógica y funcionalidades de la aplicación
- `server.js`: Servidor Express para despliegue local
- `package.json`: Configuración y dependencias
